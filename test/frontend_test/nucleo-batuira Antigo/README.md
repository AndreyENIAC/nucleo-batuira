> Solução digital desenvolvida como projeto de estágio para a **OSC Núcleo Batuíra Serviços de Promoção à Família** (Guarulhos/SP). O sistema centraliza a burocracia técnica e administrativa para otimizar o tempo da equipe multiprofissional, permitindo maior dedicação e cuidado direto aos 128 idosos acolhidos.

### 📋 Ajustes e Melhorias (Roadmap do Projeto)
O projeto cumpre os requisitos de estágio e está pronto para operação, com melhorias contínuas planejadas:
- [ ] Autenticação de usuários com troca obrigatória de senha no primeiro acesso
- [ ] Telas e Dashboards customizados por nível de acesso (Admin, Técnico, Financeiro, Funcionário)
- [ ] Módulo de Gerenciamento de Acolhidos (Cadastro, Filtros e Perfil Clínico)
- [ ] Centralização de Prontuários e Planos Assistenciais (PIA, PTS e Plano de Alta)
- [ ] Módulo Financeiro (Upload de documentos, controle de gastos e benefícios como BPC)
- [ ] Integração com banco de dados SQLite para ambiente de produção

## 🛠️ Tecnologias Utilizadas
*   **Frontend:** HTML, CSS, JavaScript, Bootstrap
*   **Backend:** Flask (Python)
*   **Banco de Dados:** SQLite
